Bem-vindo ao FinanceAI

Sistema inteligente de gestão financeira com processamento automatizado de notas fiscais

Sobre o Sistema

O FinanceAI é uma plataforma de gestão financeira que utiliza Inteligência Artificial (Google Gemini) para automatizar a extração de dados de notas fiscais e facilitar o controle de transações financeiras.

Com recursos avançados de processamento de documentos e assistente inteligente, você pode gerenciar suas finanças de forma rápida, precisa e eficiente.

Funcionalidades Principais

Importar Nota Fiscal

Faça upload de PDFs de notas fiscais e deixe a IA extrair automaticamente todos os dados:

  • Dados do fornecedor e cliente
  • Valores e datas
  • Classificações de despesas
  • Parcelas e forma de pagamento
Acessar

Assistente IA

Converse com o assistente inteligente para consultar informações financeiras:

  • Consultas em linguagem natural
  • Análises de transações
  • Relatórios personalizados
  • Histórico de conversas
Acessar

Cadastro Manual

Registre manualmente suas transações financeiras:

  • Cadastro de pessoas (fornecedor/cliente)
  • Classificações de receitas/despesas
  • Transações a pagar/receber
  • Geração automática de parcelas
Acessar

Visualizar Cadastros

Gerencie todos os registros do sistema:

  • Busca e filtros avançados
  • Edição de registros
  • Exclusão lógica (soft delete)
  • Visualização detalhada
Acessar

Como Começar

  1. Importe suas notas fiscais: Faça upload de PDFs para extração automática de dados
  2. Configure classificações: Cadastre categorias de receitas e despesas para organização
  3. Registre transações: Use o cadastro manual para adicionar movimentações financeiras
  4. Consulte com IA: Utilize o assistente para análises e relatórios inteligentes

Dica:

Use a navegação lateral para acessar rapidamente todas as funcionalidades do sistema. Cada página possui instruções específicas para facilitar o uso.
